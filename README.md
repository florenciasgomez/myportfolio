# Florencia Gomez - Portfolio

Portfolio profesional de Florencia Gomez, Senior SDET & QA Automation Engineer.

## Sobre el proyecto

Sitio web estático single-page construido con HTML, CSS y JavaScript vanilla. Incluye soporte bilingue (Español/Inglés) y diseño responsive.

### Secciones

- **Experiencia** - Trayectoria profesional en QA y testing automation
- **Skills** - Tecnologías y herramientas (Cypress, Playwright, Python, TypeScript, CI/CD, Cloud, Web3)
- **Logros** - Resultados destacados en automatización, estrategia QA e implementación de agentes de IA
- **Cursos & Certificaciones** - Formación continua en testing y tecnología
- **Proyectos** - Casos de éxito detallados con desafío, solución e impacto

### Stack

- HTML5 / CSS3 / JavaScript
- GitHub Pages (hosting)
- Sistema de internacionalización (i18n) custom
- Generación de CV en PDF desde el navegador

## Deploy

El sitio se despliega automáticamente en **GitHub Pages** desde la rama `main`.
